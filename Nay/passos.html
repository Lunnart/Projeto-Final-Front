1. clonar o repositório da luanne
  2. git checkout -b nomedabranch
  3. faço minha alteração
  4. git add .

  6. git push origin nomedabranch

  8. após aceito o PR, a gente muda de branch para atualizar
  9. git checkout main
  10. git pull
  11. agora atualizado, a gente repete o passo 2
